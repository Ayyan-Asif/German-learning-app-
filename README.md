# 🇩🇪 Wort des Tages– German Word Reminder App

Wortly is a simple and beautiful web application designed to help learners improve their German vocabulary through daily practice. The app displays German words along with their English translations and example sentences, making vocabulary learning more engaging and effective.

---

## ✨ Features

- 📖 Learn German words with English translations
- 📝 Example sentences in both German and English
- 📊 Progress tracking with a dynamic progress bar
- ✅ Mark words as "Known"
- 🔄 Browse through vocabulary with the "Next Word" button
- 📱 Fully responsive design for desktop, tablet, and mobile devices
- 🎨 Modern and clean user interface

---
